\section{Kommando 'message'}
message STRING

\subsection{look and feel}
\includegraphics{cruise_message.eps}
\subsection{Verhalten während Interpretation}
Erzeugt eine Box mit dem Textinhalt STRING.

\subsection{Verhalten bei Interaktion}
Keine Interaktion möglich

\subsection{Beispiel}

\verb*|message "Noch ein Text"|

erzeugt eine Box mit dem Inhalt 'Noch ein Text'.

\verb*|message 41+1=[expr 41 + 1]|

erzeugt eine Box mit dem Inhalt '41+1=42'
